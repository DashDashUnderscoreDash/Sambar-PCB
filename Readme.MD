# **Sambar**

Sambar is an 11u alpha stagger V4n style PCB inspired by the Subaru Sambar with leafsprings tadpole mounting

### In this repo you will find:
```
- Layout options
- Images of PCB
- Firmware
- STEP model of PCB
- Edgecuts
- Link to 3d print BEEP case
```
